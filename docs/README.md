# Karaoke Lyrics Maker — 卡拉OK歌词制作软件

## 项目简介

一款跨平台桌面端卡拉OK歌词制作软件，功能对标 Sayatoo（傻丫头字幕软件）。支持导入视频/音频，通过按键打轴（拍打节奏标记时间点），实现歌词逐字扫色效果，并导出成品视频及多种歌词文件格式。同时支持汉字注音（拼音/假名）等语言学习辅助功能。

## 核心功能一览

1. **媒体导入** — 视频文件（MP4/MKV/MOV等）和音频文件（MP3/WAV/FLAC等）
2. **歌词输入** — 手动输入、粘贴文本、导入LRC/ASS字幕文件、导入带注音的文本
3. **按键打轴** — 播放音频时通过按键（空格/自定义）标记每个字/词的起止时间
4. **逐字扫色** — 实时预览歌词扫色动画，支持多种扫色风格
5. **读音标注** — 汉字上方显示拼音、日文汉字上方显示假名，支持自动生成与手动编辑
6. **时间微调** — 波形图辅助精确定位，拖拽调整时间点
7. **样式编辑** — 字体、颜色、描边、阴影、发光、渐变等文字效果
8. **项目工程** — 保存/加载项目文件（`.klproj`），包含所有编辑数据
9. **导出成品** — 渲染带歌词的视频、导出LRC文件、导出逐字时间戳JSON
10. **跨平台** — Windows / macOS / Linux 原生体验

## 目标用户

- 卡拉OK视频制作者
- VTuber/虚拟主播字幕制作
- 歌词视频创作者（Lyric Video）
- 语言学习者（注音功能）
- 音乐教育工作者

## 非目标（不做）

- 实时录音/麦克风输入打轴（V1不考虑）
- 云端协作编辑
- 移动端（仅桌面端）
- AI自动语音识别对齐（V1不做，预留扩展接口）
- 复杂的视频剪辑功能（专注于歌词字幕制作）
